## Installation

- Install [Node.JS](https://nodejs.org/)

- run command `npm ci`

## Options

- open `.env` file
- add url value after `URL=`
- add custom delay value after `DELAY=` in seconds if you want to change it. Default: `60`
- save file

## Start

run command `npm start`

## Stop

press `Ctrl + C`
